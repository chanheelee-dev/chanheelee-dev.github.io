# Tech Stack

## Programming Language
- **Ruby**: Used for Jekyll site generation.

## Framework
- **Jekyll v4.3**: Static site generator for personal blogs and digital gardens.

## Theme
- **jekyll-theme-primer**: Minimalist theme inspired by GitHub's Primer CSS.

## Plugins
- **jekyll-feed**: RSS feed generator.
- **jekyll-seo-tag**: Optimized SEO tags for social sharing and search engines.
- **jekyll-optional-front-matter**: Support for optional front matter in content.
- **jekyll-relative-links**: Automatic relative link resolution.

## Hosting and Deployment
- **GitHub Pages**: Primary hosting for the main blog.
- **Netlify**: Used for branch preview deployments to verify changes before merging.
